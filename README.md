# CashPilot 🚀

CashPilot is a modern **startup runway calculator** built using **HTML, CSS, and JavaScript**. It helps founders, startups, and small businesses estimate how many months of cash runway remain based on available funds and monthly burn rate.

The project focuses on combining a **clean premium UI** with practical financial insights, making it feel like a real SaaS finance product rather than a basic calculator.

---

## 🌟 Features

* 💰 Input Total Cash on Hand
* 📉 Input Monthly Burn Rate
* 📊 Instant Runway Calculation (Months Remaining)
* 🟢 Safe / 🟡 Warning / 🔴 Danger Status Zones
* 📅 Estimated Zero Cash Date
* 📈 Scenario Planning Cards
* 📱 Fully Responsive Mobile-Friendly Design
* ✨ Smooth Animations & Premium UI
* ⚡ Real-Time Updates with JavaScript

---

## 📌 How It Works

The core formula used:

Runway = Total Cash on Hand ÷ Monthly Net Burn

Where:

Net Burn = Monthly Expenses - Monthly Revenue

CashPilot also allows users to test future scenarios such as growth changes, revenue improvements, and fundraising impact.
